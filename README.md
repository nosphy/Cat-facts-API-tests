# Cat-facts-API-tests
Pytest framework for testing a Cat fact retrieving API

# Test cases contained in this project

Function name | What it tests for |
--- | --- |
`test_can_call_endpoint` | check if the endpoint can be accessed |
`test_random_fact_retrieved` | check if the fact retrieved is different every time you access the API | 
`test_can_retrieve_specific_fact_by_id` | check if you can retrieve a specific fact by inputting it's ID |
`test_can_retrieve_multiple_facts` | check if you can retrieve multiple facts with one API call |
`test_can_retrieve_facts_by_language` | cehck if you can retrieve a specific fact localised in a supported language |
